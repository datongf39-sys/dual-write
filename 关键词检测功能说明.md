# Dual-Write 关键词检测功能说明

## 功能概述

在 DRAFT 编辑区下方新增了**关键词检测标签区**，可以：

1. **自动识别**顶部关键词输入框中的关键词（核心关键词和长尾关键词）
2. **实时对比** DRAFT 内容中是否使用了这些关键词
3. **高亮提醒**未使用的关键词（标红），已使用的关键词（标绿）

## 使用方法

### 1. 在顶部关键词输入框中输入关键词

格式示例：
```
核心关键词（2个）：护肤品、保湿
长尾关键词（10个）：干燥、补水、滋润、敏感肌、温和、面霜、精华、修复、舒缓、水润
```

### 2. 关键词标签会自动显示在 DRAFT 区域下方

- **红色标签**：该关键词尚未在 DRAFT 中使用
- **绿色标签**：该关键词已在 DRAFT 中使用
- **核心关键词**：字体加粗显示
- **长尾关键词**：正常字体显示

### 3. 实时更新

- 当你在关键词输入框中修改关键词时，标签会自动更新
- 当你在 DRAFT 中输入内容时，标签会实时变色（从红变绿）
- 顶部会显示统计信息：`⚠️ X 个关键词未使用` 或 `✅ 全部关键词已使用`

## 支持的格式

### 核心关键词
- `核心关键词（2个）：关键词1、关键词2`
- `核心关键词(2个):关键词1,关键词2`
- 支持中英文括号、冒号、逗号、顿号

### 长尾关键词
- `长尾关键词（10个）：关键词3、关键词4、...`
- `长尾关键词(10个):关键词3,关键词4,...`
- 支持中英文括号、冒号、逗号、顿号

## 示例

### 输入示例

在顶部关键词输入框中输入：
```
核心关键词（2个）：护肤品、保湿
长尾关键词（5个）：干燥、补水、滋润、敏感肌、温和

这款护肤品真的很好用，我是敏感肌，用了之后皮肤不再干燥了。
```

在 DRAFT 中输入：
```
这款产品非常适合敏感肌使用，保湿效果很好，能有效补水。
```

### 效果

- "保湿"、"敏感肌"、"补水" 会显示为绿色标签（已使用）
- "护肤品"、"干燥"、"滋润"、"温和" 会显示为红色标签（未使用）
- 顶部显示：`⚠️ 4 个关键词未使用`

## 注意事项

1. 关键词格式必须按照 `核心关键词（X个）：` 或 `长尾关键词（X个）：` 的格式
2. 关键词之间用逗号、顿号或中文顿号分隔
3. 如果没有检测到关键词，标签区会自动隐藏
4. 关键词匹配是精确匹配，例如"护肤"不会匹配"护肤品"
5. 标签区会自动保存到 localStorage，刷新页面后仍然保留

## 技术细节

- 关键词检测使用简单的字符串包含匹配
- 支持中英文标点符号混用
- 自动过滤空白关键词
- 实时更新，无需手动刷新
- 与现有的自动保存功能完全兼容
